Sobre o projeto

Este é um projeto simples que tem como objetivo experimentar e utilizar os conceitos iniciais que aprendi durante minha capacitação em HTML e CSS pela plataforma Alura.

Neste projeto, pude aprender mais sobre os elementos básicos do HTML e os selectors de CSS; Também pude utilizar classes customizadas; Criações de variáveis; Conceitos como: Box Model, Reset CSS, Box-Sizing e Flex Box; Importação de conteúdos externos; Organização dos arquivos de projetos; Transição entre páginas web; Responsividade de telas; Hover; Media; Entre outros.
